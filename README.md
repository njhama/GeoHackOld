# GeoHack

# Getting Started
- Download Node.js
- Download the necessary modules found in server/server.js
  - Puppeteer

# Running the Program
- Open client/index.html in a web browser, enter a port
- Cd into the server and run the server, enter the port as an argument
  - node server.js 9000
